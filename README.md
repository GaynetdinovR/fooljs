## Установка и запуск
<a id="getting_started"></a>

- Клонировать проект

```
git clone https://github.com/GaynetdinovR/fooljs.git
```

- Запустить

```
npm start
```

- Открыть в браузере

```
http://localhost:3000/
```

## Тесты
